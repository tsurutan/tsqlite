# SQLite written in TypeScript

This library is a tiny SQLite written in TypeScript, not ORM or client library.

# Install

npm
```sh
npm install --save tsqlite
```

yarn
```sh
yarn add tsqlite
```

# Repl
Please prepare the following code.


```js
const { repl } = require("tsqlite");
repl();
```
